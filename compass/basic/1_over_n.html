\(\frac{1}{n}\) of segment

Here we consider an efficient way to construct a segment of length \(\frac{1}{n}\) if we are given a segment of length 1. Suppose the segment is given as OA.

  1. Fix a length \(r\), whose exact value is negligible. Draw a straight line OB1B2...Bn, where each segment BiBi+1 has length \(r\) and the line is not parallel to OA.
  2. Connect ABn. Find C such that CB1 is parallel to ABn. OC is the desired segment of length \(\frac{1}{n}\).

For simplicity, we only show the case \(n = 3\).

The idea behind our construction is the intercept theorem.

The intercept theorem states that if CB1 is parallel to ABn and OB1 : OBn = 1:\(n\), then OC : OA = 1 : \(n\).

With this construction in mind, we can also construct any rational number efficiently.
